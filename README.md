# Option Chain Analyzer

A professional Python-based Option Chain Analyzer for traders and investors.

## Screenshots

<img src="screenshots/image.png" alt="Dashboard" width="900">

## Features

- Open Interest (OI) Analysis
- OI Change Tracking
- Put Call Ratio (PCR)
- Max Pain Calculation
- Gamma Exposure (GEX)
- Support & Resistance Levels
- Live Market Data Integration
- Trading Signals Dashboard

## Technologies Used

- Python
- PyQt5
- Pandas
- NumPy
- Plotly
- NSE Data APIs


## Installation

```bash
git clone https://github.com/ajaylkjhg-maker/option-chain-analyzer.git
cd option-chain-analyzer
pip install -r requirements.txt
python main.py
```
